# 🇫🇷 Le Petit Yaourt

A joyful French learning app for toddlers (ages 1–4).

Designed for bilingual families who want a simple, beautiful way to introduce French vocabulary through:

- 📖 Flashcard lessons
- 🎯 Quiz mode
- 🔊 Real French pronunciation
- 🖨️ Printable flashcards
- 🌙 Day & night mode

---

## 🚀 How It Works

This is a simple React app using:

- React 18 (CDN)
- ReactDOM (CDN)
- Babel (in-browser JSX transform)
- Tailwind CSS (CDN)

No build tools.
No npm.
No terminal required.

Just two files:

- `index.html`
- `app.js`
